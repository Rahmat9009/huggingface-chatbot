# AI Chatbot 🤖

A full-stack AI chatbot built from scratch as a personal learning project to understand how modern AI applications are architected and deployed.

## About

This project was my first attempt at building a complete AI-powered application end to end — from the frontend interface to the backend logic and AI integration. The goal was to go beyond just using AI tools and actually understand how they fit into a real software stack.

## Features

- 💬 Real-time AI chat interface
- 🔗 Live AI model integration
- 🖥️ Clean, responsive frontend
- ⚙️ Full-stack architecture

## Tech Stack

- **JavaScript**
- **TypeScript**
- **HTML**
- **CSS**

## What I Learned

Building this project pushed me to think seriously about how language models work, how requests and responses are structured, and how to handle errors and edge cases in AI applications. It raised questions I'm still exploring — about model limitations, response quality, and how to build AI tools that are genuinely useful for real people.

## Status

Completed — built as a personal portfolio project.
